## Interactive Elements

### Hover Effects
- **Cards**: Lift up 2px + shadow increase
- **Buttons**: Darken background + shadow
- **Links**: Color change + underline

### Transitions
- All: 0.2s ease
- Smooth, not jarring
- Consistent across UI

### Visual Feedback
- Viewed articles: 70% opacity
- Loading states: Disabled buttons
- Empty states: Helpful messages
